Инструкция по установке КИВИ Терминала на Linux ------------------------------------------------------------------- 1. Установите приложение git через Терминал: sudo apt install git 2. В Терминале пропишите команду git clone https://github.com/tix272/osmp.git 3. Откройте Проводник и в Проводнике щёлкните правой кнопкой мыши и выберите "Открыть в Терминале" 2. Далее пропишите команду 3. Потом ./maratl.sh 4. Подождите примерно 20 и готово! Для удобности нажмите на F11 для полноэкранного режима
